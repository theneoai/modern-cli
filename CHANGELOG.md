## [0.1.0] - 2026-03-17

### TL;DR

First public release of **modern-ai-cli** — a TypeScript-based CLI scaffold that brings Claude AI directly to your terminal with streaming chat, code generation, and interactive prompts.

### ✨ Features

- **`ai ask`** — single-shot Q&A with 7 prompt modes (default, code, explain, refactor, debug, creative, shell), streaming output, usage stats
- **`ai chat`** — persistent interactive session with slash commands (`/mode`, `/save`, `/load`, `/history`, `/model`, `/usage`, `/clear`, `/help`, `/exit`), auto-save every 5 turns, context-window management
- **`ai generate`** — AI-powered code/content generation with interactive type and language selection via `@clack/prompts`
- **`ai config`** — persistent JSON config with schema validation (`show`, `set`, `wizard`, `reset`, `path` subcommands)
- **`ai whatsnew`** — in-app changelog viewer and build metadata inspector (`--version-info`, `--raw`, `--count`)
- **Upgrade notifier** — background npm version check (once/24h, non-blocking, respects `CI` and `NO_UPDATE_CHECK` env vars)
- **Adaptive thinking** — claude-opus-4-6 with `thinking: { type: "adaptive" }` for reasoning-heavy requests

### 🏗️ Infrastructure

- **Build**: tsup → ESM bundle targeting Node 18+, `#!/usr/bin/env node` banner injected automatically
- **Version embedding**: `postbuild` hook writes git hash, branch, dirty flag, and build timestamp into `src/version.ts` and `dist/version-meta.json`
- **Cross-platform binaries**: `@yao-pkg/pkg` packages for Linux x64/arm64, macOS x64/arm64, Windows x64
- **CI matrix**: GitHub Actions tests across Node 18/20/22 × Linux/macOS/Windows
- **CD**: `semantic-release` pipeline with AI-enhanced release notes (fallback to conventional-changelog), automated npm publish, GitHub release with binary assets and SHA256 checksums
- **Git hygiene**: Husky v9 + lint-staged (ESLint auto-fix on pre-commit), commitlint enforcing conventional commits

### 📦 Install

```bash
# npm (recommended)
npm install -g modern-ai-cli

# Binary (no Node required)
# Linux x64
curl -L https://github.com/theneoai/modern-cli/releases/download/v0.1.0/modern-ai-cli-linux-x64.tar.gz | tar -xz
sudo mv modern-ai-cli-linux-x64 /usr/local/bin/ai

# macOS arm64 (Apple Silicon)
curl -L https://github.com/theneoai/modern-cli/releases/download/v0.1.0/modern-ai-cli-macos-arm64.tar.gz | tar -xz
sudo mv modern-ai-cli-macos-arm64 /usr/local/bin/ai
```
